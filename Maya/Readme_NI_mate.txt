Installation instructions for Delicode NI mate receiver for Maya

1. Extract this zip archive to:
   OS X:    <home folder>/Library/Preferences/Autodesk/maya
   Windows: <user’s directory>/My Documents/maya

2. Open Maya and in the MEL command line write:
   NImateReceiver<enter>

3. From the popup select the shelf to create the NI mate receiver tool to and click OK.

4. Double clicking the NI mate shelf button opens up preferences, single clicking starts and stops receiving data from NI mate.
